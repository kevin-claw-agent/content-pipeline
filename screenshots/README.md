# Screenshots

本目录用于存放项目演示截图。

## 建议截图内容

1. **homepage.png** - 首页展示
2. **demo-form.png** - 在线演示表单
3. **progress-panel.png** - 进度监控面板
4. **result-panel.png** - 生成完成界面
5. **examples-grid.png** - 示例库展示
6. **mobile-view.png** - 移动端响应式效果

## 截图规格

- 推荐分辨率：1920x1080 (桌面) / 375x812 (移动)
- 格式：PNG
- 命名规范：小写，使用连字符分隔

> 注：截图可在演示网站部署后通过浏览器开发者工具截取。
