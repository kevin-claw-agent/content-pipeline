核心功能

四大智能模块
覆盖完整生产链路

每个阶段都由专门的 AI Agent 负责，协同完成从创意到成品的全流程

📝

ScriptAgent

将故事梗概转化为完整剧本。自动规划场景结构、对话内容、旁白文案， 并根据目标时长智能调整内容密度。

  • 智能场景规划
  • 对话与旁白生成
  • 时长自适应调整
🎨

VisualAgent

基于剧本生成分镜设计。将文字转化为画面描述、镜头角度、视觉风格指导， 为视频渲染提供详细的视觉蓝图。

  • 画面描述生成
  • 镜头语言设计
  • 视觉风格统一
🎥

RenderAgent

将分镜转化为视频片段。并行渲染多个场景，自动处理画面生成、 转场效果和基础后期处理。

  • 并行视频渲染
  • 智能场景转换
  • 批量片段生成
✨

Composer

最终视频合成与输出。智能拼接片段、添加背景音乐、生成字幕、 调整节奏，输出完整成品视频。

  • 智能片段拼接
  • 音频与字幕合成
  • 成品视频输出

快速开始

简单几步
启动你的流水线

01

克隆仓库

获取项目源码到本地

02

启动依赖

Docker Compose 一键启动 PostgreSQL、Redis、MinIO

03

配置环境

复制 .env.example 并配置 API Keys

04

启动服务

运行 API 服务和 Worker 进程

Terminal 
# 克隆仓库
git clone https://github.com/kevin-claw-agent/content-pipeline.git
cd content-pipeline

# 启动依赖服务
docker-compose -f docker/docker-compose.yml up -d

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 配置你的 API Keys

# 启动 API 服务
npm run dev

# 新终端：启动所有 Worker
npm run workers

API 使用示例

# 创建 Episode
curl -X POST http://localhost:3000/episodes \
  -H "Content-Type: application/json" \
  -d '{
    "title": "我的短剧",
    "premise": "一个关于勇气的故事...",
    "targetDuration": 30
  }'

# 查询状态
curl http://localhost:3000/episodes/{episode-id}
